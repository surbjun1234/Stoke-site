# Stoke-site: 나스닥 & QQQ 대시보드

## 프로젝트 개요

`Stoke-site` 프로젝트는 나스닥 시장과 QQQ ETF의 주요 지표를 시각적으로 제공하는 웹 대시보드입니다. 이 사이트는 사용자가 시장의 전반적인 흐름과 특정 대형 기술주들의 동향을 한눈에 파악할 수 있도록 돕습니다. 특히, 과매도 상태의 주식을 필터링하는 기능을 포함하여 잠재적인 투자 기회를 탐색하는 데 유용합니다.

## 주요 기능

*   **나스닥 및 QQQ 지표 시각화**: 주요 시장 지수 및 QQQ ETF의 데이터를 직관적인 그래프와 차트로 표시합니다.
*   **과매도 주식 필터링**: 거시경제 지표와 결합하여 과매도 상태에 있는 대형주를 식별하고 목록화합니다.
*   **사용자 친화적인 인터페이스**: 깔끔하고 반응형 웹 디자인을 통해 다양한 기기에서 최적의 사용자 경험을 제공합니다.
*   **실시간 데이터 업데이트**: (예상) 백엔드 시스템 또는 API 연동을 통해 최신 시장 데이터를 반영합니다.

## 사용 기술

*   **HTML5, CSS3, JavaScript**: 웹 페이지 구조, 스타일링 및 동적 기능 구현
*   **Chart.js (예상)**: 데이터 시각화를 위한 라이브러리
*   **API 연동 (예상)**: 주식 데이터 및 거시경제 지표를 가져오기 위한 외부 API 활용

## 설치 및 실행 방법

이 프로젝트는 정적 웹 페이지로 구성되어 있으므로, 별도의 서버 설정 없이 웹 브라우저에서 `index.html` 파일을 열어 실행할 수 있습니다.

1.  **레포지토리 클론**: 
    ```bash
    git clone https://github.com/surbjun1234/Stoke-site.git
    cd Stoke-site
    ```

2.  **브라우저에서 열기**: 
    `index.html` 파일을 웹 브라우저로 드래그 앤 드롭하거나, 파일 경로를 직접 입력하여 엽니다.
    ```bash
    # 예시 (macOS)
    open index.html
    # 예시 (Windows)
    start index.html
    ```
